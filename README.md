Workforce Scheduling API

Backend do sistema de gestão de escalas e plantões, desenvolvido com Java e Spring Boot.

## Objetivo
Centralizar a criação e o gerenciamento de escalas de trabalho, garantindo regras de descanso,
evitando conflitos de horário e mantendo histórico de alterações.

## Tecnologias
- Java 17
- Spring Boot
- MySQL
- Spring Security + JWT
